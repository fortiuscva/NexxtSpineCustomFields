  This application extends the Item card to include the following fields:

- Rev Level
- Drawing Number
- Height/Depth
- Material
- System Name
- Sterile Product
- IFU Number
- FDA Device Class
- 510K Number
- Batch Prefix
- Product Family
- Product Details
- Product Type

Each field is linked to a table that users will use to store data for each record.
